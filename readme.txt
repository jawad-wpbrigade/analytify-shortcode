The basic Analytify Shortcode block

== Description ==

This package is standalone plugins and is a part of one of the beta features in Analytify Pro.

You can create all type of combinations of Analytify shortcodes with it.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/analytify-shortcode` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress
